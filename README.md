## 如何安装
```
git clone 这个项目的地址
```
```
cd 这个项目的文件
```
```
npm install
```
```
// 运行项目
// npm run docs:dev
npm run dev
```
```
// 打包
// npm run docs:build
npm run build
```

## 报错
```
控制台vue版本不匹配
npm install vue@对应的版本
```
```
网站console报错
npm install async-validator@1.11.5 --save
```
